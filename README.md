Snowy Smiles - a clinic for teeth whitening and cleaning.
#
The project was created for a real client and is available at the following link:
#
Live demo:
    - [Demo link](https://snowysmiles.org/)


Technologies used:

- React
- React DOM
- HTML5
- SCSS (modular structure)
- JavaScript (ES6+)
- BEM methodology
- Git & GitHub Pages
- Motion

Features:

- Responsive design (mobile / tablet / desktop)
- Clean and modern UI
- Component-based architecture (React)
- User-friendly navigation and page structure
- Optimized layout and semantic HTML


Getting Started:
#
1. Clone the repository:
git clone https://github.com/Vitalii120296/snowy_smiles.git
cd Snowy_smiles
#
2. Install dependencies:
npm install
# or
yarn install

3. Run the project locally:
npm start
# or
yarn start
